-f  Forks the Main Ethereum network to your local machine for development and testing. 
    Fork from another currently running Ethereum client at a given block. Input should be the HTTP 
    location and port of the other client, e.g. http://localhost:8545. You can optionally specify the 
    block to fork from using an @ sign: http://localhost:8545@1599200.

-m  or --mnemonic Use a bip39 mnemonic phrase for generating a PRNG seed, which is in turn used for 
    hierarchical deterministic (HD) account generation. Runs Ganache with an Ethereum key set based on 
    the mnemonic passed. Do not use this mnemonic anywhere other than for your tests

-i  or --networkId: Specify the network id ganache-cli will use to identify itself (defaults to the

other options 

-l or --gasLimit
    The block gas limit (defaults to 0x6691b7). Example --gasLimit=0x1fffffffffffff (Number.MAX_SAFE_INTEGER)

 --allowUnlimitedContractSize 
    Allows unlimited contract sizes while debugging. By enabling this flag, the check within the EVM 
    for contract size limit of 24KB (see EIP-170) is bypassed. Enabling this flag will cause ganache-cli 
    to behave differently than production environments.

-e  or --defaultBalanceEther: Amount of ether to assign each test account. Default is 100.
    The first 10 addresses have 100 test ETH in their balance on the local test net every 
    time you boot Ganache. 

-p  or --port: Port number to listen on. Defaults to 8545.
